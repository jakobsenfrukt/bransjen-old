# Bransjen

Website for design studio Bransjen. This repo contains only the front end, the content lives separately in Craft CMS.

**Project setup**
```
npm install
```

**Compiles and hot-reloads for development**
```
npm run serve
```

**Compiles and minifies for production**
```
npm run build
```
